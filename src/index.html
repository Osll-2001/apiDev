ALL

  • https://app-api-devf.herokuapp.com/songs

    • ID

    Traer un json con la banda con el ID indicado en la url(numerico)

  • https://app-api-devf.herokuapp.com/bands/{id}Probar

    • Traer un json con la canción con el ID indicado en la url(numerico)

  • https://app-api-devf.herokuapp.com/songs/{id}Probar

    • QUERY(S)

    Traer un json con la(s) banda(s) que coincida con el o los querys dados

    Parametros: name(texto) genre(texto) startYear(numero)

  • https://app-api-devf.herokuapp.com/bands/?name={nameBand}Probar

    • Traer un json con la(s) canción(es) que coincida con el o los querys dados

    Parametros: name(texto) band(texto) genre(texto) album(texto) releaseYear(numero) length(texto)

  • https://app-api-devf.herokuapp.com/bands/?genre={genreSong}Probar

    • POST

    Crea una nueva banda en base al request mandado(JSON)

    Cuerpo Request: name(texto) genre(texto) startYear(numero) urlImage(texto)

  • https://app-api-devf.herokuapp.com/bands -> (OBLIGATORIAMENTE ES NECESARIO EL BODY DEL REQUEST)

    • Crea una nueva canción en base al request mandado(JSON)

    Cuerpo Request: name(texto) band(texto) genre(texto) album(texto) releaseYear(numero) length(texto) lyrics(texto)

  • https://app-api-devf.herokuapp.com/songs -> (OBLIGATORIAMENTE ES NECESARIO EL BODY DEL REQUEST)

    • PUT

    Realiza una edición completa de la banda especificada en el path, el elemento a editar será especificada en el body del request(JSON)

    Elementos a editar(Todos): name(texto) genre(texto) startYear(numero) urlImage(texto)

  • https://app-api-devf.herokuapp.com/bands/{idBandToEditComplete} -> (OBLIGATORIAMENTE ES NECESARIO EL BODY DEL REQUEST)

    • Realiza una edición completa de la canción especificada en el path, el elemento a editar será especificada en el body del request(JSON)

    Elementos a editar(Todos): name(texto) band(texto) genre(texto) album(texto) releaseYear(numero) length(texto) lyrics(texto)

  • https://app-api-devf.herokuapp.com/songs/{idBandToEditComplete} -> (OBLIGATORIAMENTE ES NECESARIO EL BODY DEL REQUEST)

    • PATCH

    Realiza una edición parcial de la banda especificada en el path, el elemento a editar será especificada en el body del request(JSON)

    Elementos a editar(No todos en la misma petición): name(texto) genre(texto) startYear(numero) urlImage(texto)

  • https://app-api-devf.herokuapp.com/bands/{idBandToEditPartial} -> (OBLIGATORIAMENTE ES NECESARIO EL BODY DEL REQUEST)

    • Realiza una edición parcial de la canción especificada en el path, el elemento a editar será especificada en el body del request(JSON)

    Elementos a editar(No todos en la misma petición): name(texto) band(texto) genre(texto) album(texto) releaseYear(numero) length(texto) lyrics(texto)

  • https://app-api-devf.herokuapp.com/bands/{idBandToEditPartial} -> (OBLIGATORIAMENTE ES NECESARIO EL BODY DEL REQUEST)

    • DELETE

    Elimina a la banda con el ID indicado en la url(numerico)

  • https://app-api-devf.herokuapp.com/bands/{id}

    • Elimina a la canción con el ID indicado en la url(numerico)

  • https://app-api-devf.herokuapp.com/songs/{id}
